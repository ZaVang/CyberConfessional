# CyberConfessional V2.0 Product Spec
**Based on: docs/PRD.md**
**Date: 2026-03-30**

## 1. Core Philosophy (The Odyssey Period)
*   **Aesthetic & Tone**: Transition from a warm, supportive counseling tone to a cold, deterministic, and glitchy "digital asylum". Inspired heavily by Radiohead's *The Bends*, *OK Computer*, and *Kid A*.
*   **Vibe**: Fatalism, alienation, and mathematical determinism.

## 2. Feature Breakdown
1.  **UI Visual Refactoring (Fitter Happier / OK Computer)**:
    *   **Colors**: Remove bright/warm colors. Implement high-contrast pure black backgrounds, pale greys, dark blues, with fluorescent green and glitchy high-danger red accents.
    *   **Typography & Styling**: Terminal/ECG monitor style.
    *   **Footer/Slogan**: Replace the current warm message with `> Everything in its right place.`
    *   **Loading States**: Change to `> Karma Police Protocol Initiated...` and `> Arresting anomalies...` instead of generic loading text.
2.  **LLM Prompt Overhaul (PROMPT_GENERATE_VERDICT)**:
    *   **New Tone**: Cold, fragmented, sick poetic metaphors. Machine-like diagnostic report.
    *   **$U$ (Hidden Variables)**: Refute the user's escapism (*How to Disappear Completely*).
    *   **$Z$ (Macro Environment)**: Describe a slow, suffocating doom (*No Surprises*).
    *   **$M$ (Mediation / Self-destruction)**: Point out internal emotional collapse sharply (*Just* / *Idioteque*).
