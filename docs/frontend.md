以下是我为你构思的 **“赛博神父” (Cyber Confessional) UI/UX 设计规范**：

### 一、 全局视觉基调 (Visual Identity)

* **色彩系统 (Color Palette)：** 极度克制的暗黑模式（Deep Dark Mode）。
* **背景：** 几乎纯黑 (`#0A0A0A`)，带有极其轻微的噪点纹理，模拟老旧显像管或粗糙石墙的质感。
* **主色调 (Accent)：** 赛博霓虹十字架的颜色。推荐使用冷酷的“量子蓝” (`#00F0FF`) 或者充满警告意味的“神圣红” (`#FF003C`)。
* **文本色：** 主文本为灰白色 (`#E0E0E0`)，用于冷酷计算的数据展示使用单色荧光绿 (`#00FF41`) 配合等宽字体。


* **字体排版 (Typography)：**
* **标题与神父判词：** 选择一种带有衬线体（Serif）或碑文感的字体，营造宗教的神圣庄严感。
* **数据与引擎输出：** 强制使用等宽字体（Monospace，如 Fira Code 或 JetBrains Mono），强调冰冷的数学质感和代码属性。



### 二、 核心界面布局 (The Altar Layout)

整个应用应该是一个单页应用（SPA），不需要复杂的导航，所有的注意力都集中在“告解”这一件事上。

#### 1. 顶部：赛博祭坛 (The Header)

* **Logo：** 一个由电路板纹理或数据流组成的极简十字架，或者一个代表因果图的简单 DAG 节点图标。
* **Title：** 赛博忏悔器 (The Cyber Confessional)
* **Subtitle (微小字体)：** "In math we trust, in causality we converge." (信奉数学，归于因果)

#### 2. 中部核心区：告解室 (The Input Zone)

* 这是一个占据屏幕主视觉的巨大文本输入框。
* **UI 细节：** 输入框不要用常规的实线边框，而是用微弱的发光边框，或者是底部一条呼吸闪烁的霓虹线。
* **Placeholder：** 打字机效果轮播提示语，例如：
* *“神父，如果我当时没去那家公司……”*
* *“神父，如果我昨天按下了那个买入键……”*
* *“倾诉你的执念，让因果律为你结算结局……”*


* **提交按钮：** 放弃传统的“Submit”，改成一个带有仪式感的按钮，写着 **“进行反事实推演 (Run Counterfactual)”** 或简单的 **“告解 (Confess)”**。点击时带有一道扫描光效。

#### 3. 交互高潮：计算加载状态 (The Computation Engine)

这是整个产品最“Funny”和硬核的部分。当用户点击提交后，绝对不能只转一个无聊的菊花图（Spinner）。

* **视觉效果：** 输入框变暗，屏幕中央出现一个**动态生成的 Mermaid 因果图 (DAG)** 或者是一段滚动的**伪终端日志 (Terminal Logs)**。
* **文案模拟 (SSE 流式传输最佳)：**
* `[System] 正在连接世界线观测节点...`
* `[LLM] 提取实际干预变量 X...`
* `[LLM] 提取现实结局 Y...`
* `[Engine] 发现底层潜变量 U (极度风险厌恶)...`
* `[Engine] 初始化贝叶斯溯因模块...`
* `[Engine] 正在执行 100,000 次蒙特卡洛平行宇宙采样...`
* `[Engine] 正在应用 do(X) 干预算子...世界线开始收束...`


* **音效（可选）：** 极其微弱的服务器风扇轰鸣声或心跳声。

#### 4. 判决区：神父的裁决 (The Verdict)

加载结束后，日志折叠，神父的判词如神谕般逐字显现（Typewriter effect）。

* **排版分割：**
* **左侧/上方 (The Narrative)：** LLM 生成的带有强烈情绪按摩和毒舌属性的神父判词。
* **右侧/下方 (The Cold Hard Truth)：** 用一个充满极客感的卡片，展示后端引擎算出的**真实数据**。比如用一个醒目的红色进度条或仪表盘显示：“反事实成功率：0.0002%”，并列出反推出来的“隐变量均值”。这种感性文字与理性数据的并列，会将喜剧效果拉满。



#### 5. 底部：核心宣告 (The Footer)

按照你之前 PRD 的强烈要求，在页面最底部（固定在视口底部或内容最下方），用优雅、低对比度的灰色字体，常驻展示那段哲学文案：

> *“人总是经常后悔，总是会回想若是当初做的决定不一样，现在是什么样子。但是无论如何，过去无法改变，我们应当避免沉溺过去，而是积极展望未来。赛博神父借助因果推断的思想，帮你探索另一种情况的平行宇宙世界线是否发生了跃迁，亦或是收束到了同个结局。愿你放下执念，大步向前。”*

### 三、 交互动效 (Micro-interactions)

* **打碎重组：** 当计算出概率极低（比如 <1%）时，屏幕上的因果图可以有一个“碎裂”然后再“强行合并为一条线”的动效，隐喻“世界线的强行收束”。
* **状态切换：** 从“输入状态”切换到“审判状态”时，背景色可以发生极微弱的色温变化，比如从完全的黑变成带有一丝冰冷的幽蓝。
