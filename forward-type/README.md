# 前行者人格测试 | Forward Type

> "你不是被困在过去，你只是在整理行李。"

一个类似 MBTI/SBTI 的人格测试，核心主题是「拒绝沉溺过往，继续前行」。

## 四维度系统

| 维度 | 含义 | 低分端 (+) | 高分端 (-) |
|------|------|-----------|------------|
| **R** | 回望倾向 | R+ 翻篇快，不纠结 | R- 常回头，难放下 |
| **E** | 经验萃取 | E+ 从过去汲取智慧 | E- 吃一堑不长记性 |
| **F** | 未来引力 | F+ 被未来召唤 | F- 活在当下/过去 |
| **W** | 行动意志 | W+ 说走就走 | W- 想得多做得少 |

## 16种人格类型

### R+ 组（高回望）
- **HERO 英雄** (R+E+F+W+) - 每道伤疤都是勋章，背负过去却依然向前冲锋
- **DREAM 梦想家** (R+E+F+W-) - 在过去中寻找意义，用未来愿景治愈自己
- **VETERAN 老兵** (R+E+F-W+) - 经验丰富但困在过去，用行动麻痹自己
- **LOOP 循环者** (R+E+F-W-) - 反复咀嚼过去，陷入停滞
- **PHOENIX 凤凰** (R+E-F+W+) - 烧毁过去，浴火重生
- **SPARK 火花** (R+E-F+W-) - 想要重新开始，却迟迟不动
- **NOMAD 游牧者** (R+E-F-W+) - 穿梭时空，不在此处也不在彼处
- **DRIFT 漂流者** (R+E-F-W-) - 随波逐流，不知去向

### R- 组（低回望）
- **ARROW 箭矢** (R-E+F+W+) - 经验丰富+目标明确+行动迅猛，终极前行者
- **COMPASS 罗盘** (R-E+F+W-) - 知道方向，但还在蓄力
- **BLADE 利刃** (R-E+F-W+) - 披荆斩棘但不知道去哪
- **SEED 种子** (R-E+F-W-) - 潜力无限，还在等待破土
- **METEOR 流星** (R-E-F+W+) - 燃烧自己，冲向未来
- **KITE 风筝** (R-E-F+W-) - 被未来牵引，但需要风
- **WANDERER 旅人** (R-E-F-W+) - 一直在走，但不问去向
- **LEAF 落叶** (R-E-F-W-) - 随风飘荡，无牵无挂

## 技术特性

- **纯前端实现** - HTML + CSS + JavaScript，无依赖
- **题目随机排序** - 每次测试题目顺序不同
- **四维度打分** - 每题4个选项，对应不同维度的分数
- **Manhattan距离匹配** - 向量算法匹配16种人格原型
- **响应式设计** - 适配桌面和移动设备
- **品牌语调** - 温暖坚定，不是毒舌自嘲
